```
给定一个 N 叉树，找到其最大深度。

最大深度是指从根节点到最远叶子节点的最长路径上的节点总数。

例如，给定一个 3叉树 :
```

![img](/img/LeetCode/narytreeexample.png)

```
我们应返回其最大深度，3。

说明:

树的深度不会超过 1000。
树的节点总不会超过 5000。
```

BFS 34.79%

```java
class Solution {
    public int maxDepth(Node root) {
        if (root == null) return 0;

        Deque<Node> queue = new ArrayDeque<>();
        queue.add(root);

        int depth = 0;

        while (!queue.isEmpty()) {
            depth++;

            int size = queue.size();
            for (int i = 0; i < size; i++) {
                queue.addAll(queue.poll().children);
            }
        }

        return depth;
    }
}
```
